API

A simple REST API implemented using Flask is provided by astromlp.api. To run the API locally clone the repository and run:

$ python astromlp/api.py

By default the API listens on http://localhost:8010 and the following requests are available:

  • /infer/<model>/<objid>: request for prediction for SDSS object identifier objid using model identifier model

  • /proc/<pipeline>/<objid>: request for process an SDSS object identifier objid using pipeline identifier pipeline

Running the API using Docker

A Docker file is also available to run the API, to build the Docker image run:

$ docker build -t astromlp-api:latest .

And to run a container:

$ docker run -d --rm -p 8500:8500 astromlp-api

The API is available in http://localhost:8500, and the same methods illustrated before can be used to send queries. For example:

$ curl http://127.0.0.1:8500/infer/i2r/1237648720693755918
(...) "output":[0.09091393649578094],"x":["img"],"y":["redshift"]}
